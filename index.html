Politique de confidentialité — QuickCount

Dernière mise à jour : mars 2026

QuickCount est une application mobile de comptage personnel. Cette politique explique comment vos données sont traitées.

Données collectées

QuickCount ne collecte aucune donnée personnelle identifiable. Les seules données stockées sont vos enregistrements de comptage (nombre, date, heure), conservées uniquement sur votre appareil.

Partage de données

Aucune donnée n'est transmise à des tiers. Aucune publicité. Aucun suivi analytique.

Suppression des données

Vous pouvez supprimer toutes vos données depuis Paramètres → Réinitialiser toutes les stats, ou en désinstallant l'application.

Contact

Pour toute question, contactez-nous à : votre-email@domain.com
